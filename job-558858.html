ר״צ פיתוח תוכנה (Team Lead)

מיקום: פתח תקוה · קוד משרה: #558858

לחברה פיננסית בפתח תקוה דרוש/ה ראש צוות להובלת CTO Infrastructure Application Data Team. התפקיד משלב הובלה טכנולוגית עם ניהול אנשים: חלק מהזמן פיתוח Hands-on וחלק חניכה וניהול צוות. עבודה צמודה עם Product, QA, DevOps ו-Project Management, הובלת פרויקטים חדשניים בענן ובבינה מלאכותית גנרטיבית (Generative AI).

תחומי אחריות:
  • פיתוח ותחזוקה של מערכות סקיילביליות וביצועיסטיות ב-Python / Java לפי צורך.
  • איכות קוד, אבטחה ועמידה בסטנדרטים מקצועיים.
  • הנעת תהליכי Agile, שיפור CI/CD ואופטימיזציה של זרימות עבודה.
  • שיתוף פעולה חוצה-ארגון ליישור פתרונות טכניים עם היעדים העסקיים.
  • מחקר והטמעת טכנולוגיות חדשות (ענן/GenAI) לשיפור יעילות.
דרישות:
  • ניסיון בניהול/חניכת צוותי פיתוח (2+ שנים).
  • 6+ שנות פיתוח מערכות גדולות ב-Java ו-Spring Boot.
  • מומחיות בענן AWS, ארכיטקטורת Microservices וקונטיינרים (Docker, Kubernetes).
  • ניסיון פרונט-אנד עם React / Angular / Vue.
  • Backend: מסדי נתונים SQL ו-NoSQL, ORM (כגון Hibernate), אינטגרציית APIs.
  • CI/CD ו-IaC: Terraform / CloudFormation, ומתודות DevOps.
  • יתרון: שפה נוספת כגון Python / Node.js / JavaScript.
English (original) – short

We’re seeking a Team Leader to head our CTO Infrastructure Application Data Team. The role blends hands-on development with people leadership. You’ll drive cloud and Generative AI initiatives, collaborate cross-functionally, and deliver scalable, secure solutions.
