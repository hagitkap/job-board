מפתח/ת Delphi מנוסה

מיקום: בני ברק (ליד מגדלי בסר) · מודל: היברידי (יומיים מהבית) · קוד משרה: #636754

לחברה וותיקה ומובילה בתחום השכר והפיננסים דרוש/ה מפתח/ת Delphi עם לפחות 5 שנות ניסיון לפיתוח ותחזוקת מערכות חיות המשרתות אלפי לקוחות. עבודה בצוות טכנולוגי מתקדם, אתגר מקצועי גדול וסביבה דינמית.

Delphi SQL Reports Hybrid
תחומי אחריות:
  • פיתוח ותחזוקת יישומים קיימים בסביבת Delphi.
  • כתיבת שאילתות SQL מורכבות מול בסיסי נתונים (Access / Paradox).
  • פיתוח והטמעת דוחות באמצעות QuickReport / FastReport.
  • פתרון תקלות, ייעול ביצועים ושיפור יציבות.
  • עבודה שוטפת עם צוותי QA, מוצר ו־IT.
דרישות:
  • תואר ראשון במדעי המחשב / הנדסת תוכנה – חובה.
  • 5+ שנות ניסיון עדכני בפיתוח ב־Delphi כשפה עיקרית – חובה.
  • ניסיון בכתיבת שאילתות SQL מורכבות – חובה.
  • ניסיון עם QuickReport או FastReport – יתרון.
  • ניסיון מול Access ו/או Paradox – יתרון.
  • ניסיון ב־C# / JSON / XML / עבודה עם API – יתרון.
כישורים נוספים:
  • יכולת עבודה עצמאית ובצוות, חשיבה אנליטית ופתרון בעיות.
  • תקשורת בין־אישית טובה ויכולת למידה עצמית גבוהה.

היקף: משרה מלאה, א׳–ה׳, 08:30–17:00, היברידי (יומיים מהבית).
